**FAQ Chatbot using NLP**

Description:
This project is a simple FAQ chatbot that answers user questions by matching them with a predefined list of frequently asked questions. It uses basic Natural Language Processing (NLP) techniques and cosine similarity to find the most relevant answer and return it to the user.

Technologies: Python, NLTK/SpaCy, Scikit-learn

**Real-Time Object Detection and Tracking using DroidCam**

Description:
This project performs real-time object detection and tracking using a mobile camera connected through DroidCam. It uses the YOLOv8 model to detect objects and the SORT algorithm to track them with unique IDs in live video.

Technologies: Python, OpenCV, YOLOv8, SORT, DroidCam
